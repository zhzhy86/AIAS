### 车辆检测SDK
车辆检测是城市交通监控中非常重要并且具有挑战性的任务，该任务的难度在于对复杂场景中相对较小的车辆进行精准地定位和分类。
支持car (汽车)，truck (卡车)，bus (公交车)，motorbike (摩托车)，tricycle (三轮车)等车型的识别。

### SDK功能
- 车辆检测，给出检测框和置信度
- 支持的类别：
- car
- truck
- bus
- motorbike
- tricycle
- carplate
