### 行人检测SDK
行人检测是利用计算机视觉技术判断图像中是否存在行人并给予精确定位，一般用矩形框表示。
行人检测技术有很强的使用价值，它可以与行人跟踪，行人重识别等技术结合，应用于汽车无人驾驶系统，
智能视频监控，人体行为分析，客流统计系统，智能交通等领域。


### SDK功能
- 行人检测，给出检测框和置信度
