### 风格迁移(预定义4种)SDK
风格迁移可以把一张图片转换成另一种风格。本sdk预定义了4种画风：
- 塞尚(Paul Cezanne, 1838～1906)
- 莫奈 (Claude monet, 1840～1926)
- 日本浮世绘
- 梵高 (Vincent Willem van Gogh, 1853~1890)


- 原图.       
![img](https://aias-home.oss-cn-beijing.aliyuncs.com/AIAS/gan_sdks/scenery.jpeg)
