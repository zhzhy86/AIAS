![result](https://aias-home.oss-cn-beijing.aliyuncs.com/AIAS/video_sdk/rtsp2rtmp.png)

## 运行例子
1. 首先下载例子代码
```bash
git clone https://github.com/mymagicpower/AIAS.git
```

2. 导入examples项目到IDE中：
```
cd live_stream_sdks/rtsp2rtmp_sdk
```

3. 运行例子代码：LiveStream2MP4
