## SDK功能
mp4文件转gif动态图。

## 运行例子
1. 首先下载例子代码
```bash
git clone https://github.com/mymagicpower/AIAS.git
```

2. 导入examples项目到IDE中：
```
cd live_stream_sdks/mp4togif_sdk
```

3. 运行例子代码：Mp4ToGif


## 效果如下：
录制gif保存于
```
build/output/result.gif
```

生成gif效果图：
![gif](https://aias-home.oss-cn-beijing.aliyuncs.com/AIAS/video_sdk/result.gif)
