## 目录：
http://aias.top/

## SDK功能
rtsp/rtmp 拉流连续截图。
图片名称模板“image-%03d.jpg”，使用这个模板名称生成的图片文件：
image-001.jpg
image-002.jpg
image-003.jpg
image-004.jpg
...

## 运行例子
1. 首先下载例子代码
```bash
git clone https://github.com/mymagicpower/AIAS.git
```

2. 导入examples项目到IDE中：
```
cd live_stream_sdks/live2images_sdk
```

3. 运行例子代码：LiveStream2Images


## 效果如下：
录制图片保存于
```
build/images/
```
